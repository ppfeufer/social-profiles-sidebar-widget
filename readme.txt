Add a sidebarwidget for social profiles in you blog. Supports several sets of icons and three different iconsizes. 32, 64, and 128 Pixel.

== Description ==

This plugins installs a sidebar widget to show you social profile-icons.

**Features**

* Easy install
* Panel for easy configuration
* Sveral Icon Sets available

**Languages**

* German
* English
* Serbo-Croatian

== Installation ==

You can use the built in installer and upgrader, or you can install the plugin manually.

**Installation via Wordpress**

1. Go to the menu 'Plugins' -> 'Install' and search for 'Social Profiles Sidebar Widget'
1. Click 'install'

**Manual Installation**

1. Upload folder `social-profiles-sidebar-widget` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure plugin in your widget-settings
